<h1>Programa em Java para calculo de notas e status de aprovação de aluno</h1>
<p>O código define a classe Student, que representa um aluno com nome e três notas trimestrais. Cada trimestre tem um peso específico: </p>

<p>Primeiro trimestre: 30 pontos</p>
<p>Segundo trimestre: 35 pontos</p>
<p>Terceiro trimestre: 35 pontos</p>
<p>A classe possui os seguintes métodos:</p>

<p>notes(): soma as três notas do aluno.</p>
<p>result(): Verifica se a nota final atinge pelo menos 60% do total (60 pontos). Se sim, exibe "PASS". Caso contrário, exibe "FAILED" e informa quantos pontos faltam para alcançar o mínimo necessário para aprovação.</p>
<p>Esse código permite calcular a nota final de um aluno e determinar se ele foi aprovado ou reprovado com base nos critérios estabelecidos.</p>
